# Install Twellie on a 2014 H-series Orsay TV from Windows

1. Download the installer for [⬇️ 64-bit Windows](/releases/latest/download/twellie-orsay-host-windows-x64.zip) or [⬇️ 32-bit Windows](/releases/latest/download/twellie-orsay-host-windows-x86.zip).
2. Extract the zip file on the PC.
3. Double-click `Install-Twellie.bat`.
4. If Windows Firewall asks, allow access on private networks.
5. Leave the installer window open and note the IP address it prints.
6. On the TV, press **Menu**.
7. Go to **Smart Hub** -> **Samsung Account** -> **Log In**.
8. Enter `develop` as the account name, enter `000000` as the password, and confirm the login.
9. Open **Smart Hub**.
10. Highlight any installed app.
11. Press and hold **OK/Enter** for about 5 seconds.
12. Choose **IP Setting**.
13. Enter the IP address from step 5 and confirm.
14. Highlight any installed app again.
15. Press and hold **OK/Enter** for about 5 seconds.
16. Choose **Start User App Sync**.
17. Wait for the TV to show **Service installed**.
18. Open **Twellie** from Smart Hub.
19. Close the installer window on the PC.
