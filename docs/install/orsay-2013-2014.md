# Install Twellie on an Orsay TV (2013–2014, F & H series)

For Samsung Smart TVs from **2013 (F)** and **2014 (H)** — the 5th character of
the model number is the year letter, e.g. `UN55`**`F`**`8000` (2013) or
`UN55`**`H`**`6350` (2014).

> Twellie is an **unofficial** viewer for Twitch. Not affiliated with Twitch,
> Samsung, or Tizen.

These TVs can't make a modern secure (TLS/SNI) connection to Twitch, so you run
a tiny **host** app on a computer on the same Wi-Fi. It installs Twellie onto
the TV and relays Twitch for it while you watch.

## You need

- [**Download `twellie-orsay-host.zip`**](https://github.com/nikitakatchik/smarttv-twitch/releases/latest/download/twellie-orsay-host.zip) — always points to the latest release.
- [Node.js 18+](https://nodejs.org) on a computer (Mac / Windows / Linux) on the
  **same Wi-Fi** as the TV.

## Steps

1. **Run the host.** Unzip `twellie-orsay-host.zip`, open a terminal in that folder,
   and run:
   ```bash
   node tools/host.js
   ```
   It prints your computer's IP and the exact steps below. Leave it running.

2. **Install on the TV — choose one:**

   **A) Over the network (recommended)**
   1. On the TV open **Smart Hub** and sign in with the account name
      **`develop`** — type the word *develop*; it's a username, **not** your
      email (any 6-or-more-character password works).
   2. Open the developer / IP setting and set the **App Sync server IP** to the
      IP the host printed.
   3. Choose **Start App Sync** (a.k.a. *Start User App Synchronization*).
      Twellie installs into Smart Hub.

   **B) USB stick**
   1. Format a USB stick as **FAT32**.
   2. Create a folder named **`userwidget`** at its root.
   3. Copy the host's printed `dist/host/widget` folder inside it.
   4. Plug it into the TV — it imports automatically.

3. **Open Twellie** and watch. **Keep the host window open** while watching — the
   TV streams Twitch through it.

## Troubleshooting

- **The `develop` login fails** — it's the literal username `develop`, not your
  email address.
- **The TV can't reach the host** — confirm both are on the same Wi-Fi, allow the
  port through your computer's firewall, and use the exact IP the host printed.
- **The TV's IP field won't take a port** — some firmware assumes port 80. Run
  `node tools/host.js 80` (may require admin/sudo).
- **A "Commercial break" plays first** — anonymous playback gets Twitch's
  pre-roll ad; it clears to the live stream after a few seconds.
- **Playback stutters** — lower the quality in the player with CH▲ / CH▼.
