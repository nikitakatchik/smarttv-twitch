# Install Twellie on an Orsay TV (2013–2014, F & H series)

For Samsung Smart TVs from **2013 (F)** and **2014 (H)** — the 5th character of
the model number is the year letter, e.g. `UN55`**`F`**`8000` (2013) or
`UN55`**`H`**`6350` (2014).

> Twellie is an **unofficial** viewer for Twitch. Not affiliated with Twitch,
> Samsung, or Tizen.

These TVs sideload apps over your Wi-Fi from a computer, so you run a small
**installer** once. It pushes Twellie onto the TV; then you **close it** and the
TV streams Twitch on its own. The installer is **self-contained** (it bundles a
signed copy of Node) — nothing to install.

> Older **2011–2012** (D/E) sets aren't supported — their MAPLE engine and aged
> TLS make them unreliable.

## You need

- The installer for your computer — each link always points at the latest release:
  - macOS (Apple Silicon, M1–M4): [**twellie-orsay-host-macos-arm64.zip**](https://github.com/nikitakatchik/smarttv-twitch/releases/latest/download/twellie-orsay-host-macos-arm64.zip)
  - macOS (Intel): [**twellie-orsay-host-macos-x64.zip**](https://github.com/nikitakatchik/smarttv-twitch/releases/latest/download/twellie-orsay-host-macos-x64.zip)
  - Windows (64-bit): [**twellie-orsay-host-windows-x64.zip**](https://github.com/nikitakatchik/smarttv-twitch/releases/latest/download/twellie-orsay-host-windows-x64.zip)
- That computer on the **same Wi-Fi** as the TV.

## Steps

1. **Start the installer.** Unzip the download — the folder is self-contained, so
   there's nothing to install.
   - **macOS:** double-click **`Install-Twellie.command`**. The first time, macOS
     asks about the launcher script (it isn't signed) — right-click it, choose
     **Open**, then **Open**. The launcher clears the "downloaded from the
     internet" quarantine flag on the folder and starts. (The bundled Node is
     Developer-ID-signed by the Node.js Foundation but not separately notarized,
     so a freshly-downloaded copy would otherwise be blocked — clearing the flag
     is what lets it run.)
   - **Windows:** double-click **`Install-Twellie.bat`** (the bundled `node.exe`
     is signed, so there's no warning).

   It prints your computer's IP. Leave it running for the next step.

2. **Install on the TV** (network App-Sync — the reliable method on F/H):
   1. On the TV open **Smart Hub** and sign in with the account name
      **`develop`** — type the word *develop*; it's a username, **not** your
      email (any 6-or-more-character password works).
   2. Open the developer / IP setting and set the **App Sync server IP** to the
      IP the installer printed.
   3. Choose **Start App Sync** (a.k.a. *Start User App Synchronization*).
      Twellie installs into Smart Hub.

   > USB widget install (`/userwidget`) is disabled on most F/H firmware — use
   > the network method above.

3. **Open Twellie** and watch. Once it's installed you can **close the installer**
   — the TV streams Twitch directly.

> Most F/H sets reach Twitch directly. If browsing or playback fails right after
> a clean install, your panel's firmware likely can't negotiate modern TLS to
> Twitch — those sets aren't supported.

## Troubleshooting

- **macOS blocks the launcher** — that's the unsigned `Install-Twellie.command`
  script; right-click it → **Open** → **Open**. (Running `./node host.js`
  directly on a fresh download fails instead — Gatekeeper kills the un-notarized
  Node; the launcher works because it strips the quarantine flag first. If you'd
  rather use Terminal, run `xattr -dr com.apple.quarantine .` once, then
  `./node host.js`.)
- **The `develop` login fails** — it's the literal username `develop`, not your
  email address.
- **The TV can't reach the installer** — confirm both are on the same Wi-Fi, allow
  the port through your computer's firewall, and use the exact IP it printed.
- **The TV's IP field won't take a port** — some firmware assumes port 80. Start
  on port 80 by passing it as an argument: `./node host.js 80` (macOS, may need
  `sudo`) or `node.exe host.js 80` (Windows, may need an admin prompt).
- **A "Commercial break" plays first** — anonymous playback gets Twitch's
  pre-roll ad; it clears to the live stream after a few seconds.
- **Playback stutters** — lower the quality in the player with CH▲ / CH▼.
