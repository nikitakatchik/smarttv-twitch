# Install Twellie with TizenBrew

Use this guide for supported Samsung Tizen TVs from 2017 and newer when you want
to run Twellie as a TizenBrew module. This path depends on TizenBrew, Media
Source Extensions, and the TV web runtime, so model-specific reports are welcome.


0. If you don't have **TizenBrew** installed on your TV, get it [here](https://github.com/reisxd/TizenBrew) first.
1. Open **TizenBrew** on the TV.
2. Open **Module Manager**.
3. Choose **Add GitHub module**.
4. Search for or enter `nkatchik/smarttv-twitch`.
5. Confirm the add, then return to **Module Manager**.
6. Select **Twellie**.
