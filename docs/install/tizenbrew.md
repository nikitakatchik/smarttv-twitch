# Install Twellie with TizenBrew

Use this guide for supported Samsung Tizen TVs from 2017 and newer when you want
to run Twellie as a TizenBrew module. This path depends on TizenBrew, Media
Source Extensions, and the TV web runtime, so model-specific reports are welcome.

1. Download [⬇️ TizenBrew Installer](https://github.com/reisxd/TizenBrewInstaller/releases/latest) to your computer.
2. Connect the computer and the TV to the same network.
3. On the TV, press **Home** or **Smart Hub**.
4. Open **Apps**.
5. Enter **1 2 3 4 5** on the remote. If the remote has no number buttons, press **123** and use the on-screen keypad.
6. In the **Developer Mode** popup, set **Developer mode** to **On**.
7. Enter your computer's local IP address as **Host PC IP**.
8. Select **OK**.
9. Restart the TV. A full power-off and power-on is fine.
10. Run **TizenBrew Installer** on the computer.
11. Click **Install TizenBrew**.
12. Sign in to your Samsung account if TizenBrew Installer asks.
13. On the TV, open **TizenBrew**.
14. Open **Module Manager**.
15. Choose **Add GitHub module**.
16. Search for or enter `nkatchik/smarttv-twitch`.
17. Confirm the add, then return to **Module Manager**.
18. Select **Twellie**.
