# Install Twellie with TizenBrew

Use this guide for supported Samsung Tizen TVs from 2017 and newer when you want
to run Twellie as a TizenBrew module. This path depends on TizenBrew, Media
Source Extensions, and the TV web runtime, so model-specific reports are welcome.

1. Download [⬇️ Twellie TizenBrew package](https://github.com/nkatchik/smarttv-twitch/releases/latest/download/twellie-tizenbrew.zip) to your computer.
2. Copy `twellie-tizenbrew.zip` to a USB drive.
3. Download [⬇️ TizenBrew Installer](https://github.com/reisxd/TizenBrewInstaller/releases/latest) to your computer.
4. Connect the computer and the TV to the same network.
5. On the TV, press **Home** or **Smart Hub**.
6. Open **Apps**.
7. Enter **1 2 3 4 5** on the remote. If the remote has no number buttons, press **123** and use the on-screen keypad.
8. In the **Developer Mode** popup, set **Developer mode** to **On**.
9. Enter your computer's local IP address as **Host PC IP**.
10. Select **OK**.
11. Restart the TV. A full power-off and power-on is fine.
12. Run **TizenBrew Installer** on the computer.
13. Click **Install TizenBrew**.
14. Sign in to your Samsung account if TizenBrew Installer asks.
15. Plug the USB drive into the TV.
16. On the TV, open **TizenBrew**.
17. Open **Module Manager**.
18. Choose **Install from local package**.
19. Select `twellie-tizenbrew.zip`.
20. Select **Twellie** in TizenBrew.
