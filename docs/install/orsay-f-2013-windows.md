# Install Twellie on a 2013 F-series Orsay TV from Windows

1. Download the installer for [⬇️ Windows (64-bit)](/releases/latest/download/twellie-orsay-host-windows-x64.zip) or [⬇️ Windows (32-bit)](/releases/latest/download/twellie-orsay-host-windows-x86.zip).
2. Extract the zip file on the PC.
3. Double-click `Install-Twellie.bat`.
4. If Windows Firewall asks, allow access on private networks.
5. Leave the installer window open and note the IP address it prints.
6. On the TV, press **Menu**.
7. Go to **Smart Features** -> **Samsung Account** -> **Log In**.
8. Enter `develop` as the account name and confirm the login.
9. Open **Smart Hub**.
10. Open **More Apps**.
11. Open **Options** in the top-right corner.
12. Choose **IP Setting**.
13. Enter the IP address from step 5 and confirm.
14. Open **Options** again.
15. Choose **Start App Sync**.
16. Wait for the TV to show **Service installed**.
17. Open **Twellie** from Smart Hub.
18. Close the installer window on the PC.
