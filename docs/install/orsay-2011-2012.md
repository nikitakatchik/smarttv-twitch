# Install Twellie on an Orsay TV (2011–2012, D & E series)

For Samsung Smart TVs from **2011 (D)** and **2012 (E)** — the 5th character of
the model number is the year letter, e.g. `UN55`**`D`**`8000` (2011) or
`UN55`**`E`**`8000` (2012).

> Twellie is an **unofficial** viewer for Twitch. Not affiliated with Twitch,
> Samsung, or Tizen.

These are the oldest supported TVs. They can't make a modern secure (TLS/SNI)
connection to Twitch, so you run a tiny **host** app on a computer on the same
Wi-Fi. It installs Twellie onto the TV and relays Twitch for it while you watch.

> **Note on 2011 (D series):** D sets run the old MAPLE engine — the slowest,
> least capable target. Expect lower quality and the occasional reconnect, and
> use a big always-live channel to test first.

## You need

- [**Download `twellie-orsay-host.zip`**](https://github.com/nikitakatchik/smarttv-twitch/releases/latest/download/twellie-orsay-host.zip) — always points to the latest release.
- [Node.js 18+](https://nodejs.org) on a computer (Mac / Windows / Linux) on the
  **same Wi-Fi** as the TV.

## Steps

1. **Run the host.** Unzip `twellie-orsay-host.zip`, open a terminal in that folder,
   and run:
   ```bash
   node tools/host.js
   ```
   It prints your computer's IP and the exact steps below. Leave it running.

2. **Install on the TV — USB is the most reliable on D/E firmware:**

   **A) USB stick (recommended for 2011–2012)**
   1. Format a USB stick as **FAT32**.
   2. Create a folder named **`userwidget`** at its root.
   3. Copy the host's printed `dist/host/widget` folder inside it.
   4. Plug it into the TV — it imports automatically.

   **B) Over the network**
   1. Open the TV's smart menu (called **Internet@TV** on some 2011/2012 sets)
      and sign in with the account name **`develop`** — type the word *develop*;
      it's a username, **not** your email (any 6+ character password).
   2. Find the developer / IP setting and set the **App Sync server IP** to the
      IP the host printed.
   3. Choose **Start App Sync**. Twellie installs.

3. **Open Twellie** and watch. **Keep the host window open** while watching — the
   TV streams Twitch through it.

## Troubleshooting

- **The `develop` login fails** — it's the literal username `develop`, not your
  email address.
- **The developer/App-Sync screens are hard to find** — they're buried deeper on
  D/E firmware; the USB `userwidget` method above avoids them entirely.
- **The TV can't reach the host** — confirm both are on the same Wi-Fi, allow the
  port through your computer's firewall, and use the exact IP the host printed.
- **Nothing plays on a 2011 (D) set** — it's the weakest engine; try a big,
  always-live channel and a lower quality (CH▲ / CH▼) first.
- **A "Commercial break" plays first** — anonymous playback gets Twitch's
  pre-roll ad; it clears after a few seconds.
